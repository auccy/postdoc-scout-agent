## Portfolio Description

Postdoc Scout Agent is a Python research software MVP that turns postdoctoral supervisor discovery into an auditable evidence pipeline. It maps institution ecosystems, builds source-specific queries, collects scholarly evidence, extracts author clusters, ranks candidate supervisors with transparent scoring, and enriches profiles with preliminary funding and author evidence. The project emphasizes translational digital medicine and clinical AI rather than pure method-heavy profiles.
