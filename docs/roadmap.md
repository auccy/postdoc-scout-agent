# Roadmap

## v0.1.0

- Python package skeleton.
- CLI entry point.
- U.S. curated seed map and validation.
- Institution ecosystem mapper.
- Discovery query builder.
- OpenAlex and PubMed publication evidence connectors.
- Author extraction and candidate clustering.
- Candidate ranking and scoring.
- Candidate profile enrichment.
- Full pipeline orchestrator.
- GitHub documentation, demo assets, and CI.

- Improve seed-map verification workflows.
- Add more robust author disambiguation.
- Add additional evidence connectors with mocked tests.
- Improve ranking calibration using transparent examples.
- Add richer demo artifacts and review packets.

## Later

- Add lab-page and opening-signal workflows with explicit verification.
- Add optional LLM-assisted summaries after provenance is stable.
- Add a UI for reviewing evidence and rankings.
- Add export formats for application planning and supervisor outreach review.
